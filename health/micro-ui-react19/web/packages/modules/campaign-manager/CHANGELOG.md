# Changelog

## 1.0.0 - 2026-02-03

#### Admin Console

- **React 19 Migration**
  Upgraded to React 19 with updated dependencies and compatibility fixes.

- **New Campaign Home Page**
  Introduced a centralized campaign home page with quick actions for creating campaigns and accessing templates.

- **Campaign Templates Feature**
  Added support for campaign templates allowing users to create campaigns from pre-configured templates.

- **Redesigned App Configuration System**
  Completely redesigned app configuration with Redux state management, drag-and-drop field support, and live mobile preview.

- **New Campaign Creation Flow**
  Streamlined campaign creation with dedicated pages for Campaign Details, App Modules, App Features, and Unified Upload Screen.

- **Localization Management**
  Added localization management feature for configuring multi-language support within campaigns.

- **New My Campaigns Page**
  Redesigned My Campaigns listing page with improved filtering and campaign status tracking.

- **Rich App Configuration Components**
  Added multiple reusable templates including Button, Card, Switch, Panel, ListView, Table, Dropdown, RadioList, and Expandable components.

- **Unified Console Sheet**
  Introduced unified console sheet feature for consolidated data upload and download with multi-sheet Excel support.

- **Advanced Properties for Template Components**
  Enhanced template components with advanced property configuration including styling, visibility rules, and conditional rendering options.

- **Dependency Fields Enhancements**
  Improved dependency field logic with support for complex conditional expressions, multi-field dependencies, and cascading field updates.

- **Navigation Logic Enhancements**
  Enhanced navigation logic system with support for conditional page flows, dynamic routing based on field values, and skip logic configuration.

- **Form Configuration Enhancements**
  Improved form configuration capabilities with validation rules, field grouping, repeatable sections, and dynamic form generation.

- **Create Campaign Flow Enhancements**
  Additional features in campaign creation including boundary selection improvements, date range validations, cycle configuration, and delivery rule setup.

- **Bug Fixes**
  - Fixed spread operator priority issues in configuration merging
  - Fixed MDMS schema validation for selection fields
  - Fixed locale issues in unified console sheet download
